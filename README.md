# Real Estate Advertising Website

This project is a **Real Estate Advertising Website** that allows users to view listings of houses for sale or rent, including a gallery of images for each property. The website is built using **HTML** and **SCSS** for a clean, modern, and responsive design.

## Live Demo

You can view the live version of the website at:

[Live Website](https://codzillas.github.io/real-estate-advertising/)

## Features

-   **Property Listings**: Display property details such as price, location, size, and more.
-   **Image Gallery**: Each property listing has a gallery of images that users can browse.
-   **Responsive Design**: The website is optimized for both desktop and mobile users.

## Technologies Used

-   **HTML5**: To structure the website content.
-   **SCSS (Sassy CSS)**: For more efficient and organized styling.

## Installation & Setup

To run this project locally, follow these steps:

1. **Clone the repository**:

```
git clone https://github.com/zeynarang/real-estate-advertising
```

2. **Install required packages**:

```
npm i
```

3. **Start Dev Server**:

```
npm start
```

**To build the project we can use**:

```
npm build
```
